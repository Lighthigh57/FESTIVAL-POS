# 🎪 学校祭POSシステム

Windows PCをサーバーとして、iPadのブラウザから操作する決済システムです。

---

## 📋 必要なもの

- **PC（サーバー機）**: Windows 11
  - Node.js (https://nodejs.org) のインストールが必要
- **iPad**: 各出店2台（客用・店員用）
- **ネットワーク**: PC・iPad が同じWi-Fiに接続していること

---

## 🚀 起動方法

### 1. Node.js のインストール（初回のみ）
1. https://nodejs.org にアクセス
2. 「LTS」版をダウンロードしてインストール

### 2. サーバーの起動
1. `起動.bat` をダブルクリック
2. 自動的に必要なパッケージがインストールされ、サーバーが起動します
3. 表示されるIPアドレスを確認してください（例: `192.168.1.10`）

### 3. 管理画面で出店を作成
PC のブラウザで管理画面を開く:
```
http://localhost:3000/admin
```
- 出店ID（例: `takoyaki`）と出店名を入力して「作成する」
- 商品管理タブで商品名・価格・在庫数を設定して保存

### 4. iPadでアクセス
出店ごとに、iPadのSafariで以下にアクセス:

| 画面 | URL |
|------|-----|
| 客用iPad | `http://[PCのIP]:3000/customer?store=[出店ID]` |
| 店員用iPad | `http://[PCのIP]:3000/staff?store=[出店ID]` |

例: PCのIPが `192.168.1.10`、出店IDが `takoyaki` の場合:
- 客用: `http://192.168.1.10:3000/customer?store=takoyaki`
- 店員用: `http://192.168.1.10:3000/staff?store=takoyaki`

管理画面の「URLコピー」ボタンを使うと便利です。

---

## 💡 使い方

### 客側iPad の操作
1. 商品ボタンをタップして選択（在庫数がリアルタイム表示）
2. 右のカートで数量を確認
3. 「決済へ進む」ボタンをタップ
4. **PayPay** または **現金** を選択
   - PayPay: QRコードが表示される → スキャンして送金
   - 現金: 「店員にお支払いください」画面が表示される

### 店員側iPad の操作
1. 客が支払い方法を選択すると「決済が完了した」ボタンが出現
2. 現金を受け取った / PayPay送金を確認したら「決済が完了した」をタップ
3. 売上・在庫が自動更新される

### 管理画面（PC）
- 売上データの確認・集計
- 在庫数のリセット・商品の追加・削除
- 各出店のURLの確認

## ⚠️ PayPayについて

現在のQRコードは**デモ用URL**です。実際のPayPay送金を行うには:

**方法A: PayPay受け取りリンクを事前に作成**
1. PayPayアプリ → 「受け取る」→ 金額指定せず受け取りリンクを作成
2. そのQRコード画像をiPadの壁紙にする（金額は口頭で確認）

**方法B: 金額固定QR（要法人PayPayアカウント）**
- 法人向けPayPayが必要なため、個人では難しい

**推奨**: 事前にPayPay個人アカウントの「受け取りQR」を印刷して張り出し、
システムはその案内表示のみに使う方法が最も確実です。

## 🔧 トラブルシューティング

| 症状 | 対処法 |
|------|--------|
| iPadからアクセスできない | PCとiPadが同じWi-Fiに接続しているか確認 |
| 「接続中...」が消えない | サーバーが起動しているか確認。PCのファイアウォールでポート3000を許可 |
| QRコードが表示されない | インターネット接続を確認（QRライブラリの読み込みに必要） |

### Windowsファイアウォールの設定
コントロールパネル → Windowsファイアウォール → 詳細設定
→ 受信の規則 → 新しい規則 → ポート → TCP 3000 → 許可
